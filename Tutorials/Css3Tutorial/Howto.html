CSS How To...

When a browser reads a style sheet, it will format the HTML document accoding to the information in the style sheet.

Three Ways to Insert CSS

There are three ways of inserting a style sheet:

  • External style sheet
  • Internal style sheet
  • Inline style

External Style Sheet

With an external style sheet, you change change the look of an entire website by changing just one file!

Each page must include a reference to the externa style sheet file inside the <link> element. The <link> element goes inside the <head> section:

Example

<head> {
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>
}

An external style sheet can be written in any text editor. The file should not contain any HTML tags. The style sheet file must be daved with a .css extension.
