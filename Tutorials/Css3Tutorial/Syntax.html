CSS Syntax and Selectors

CSS Syntax

A CSS rule-set consists of a selector and declaration block:

The selector points to the HTML element you want to style.

The declaration block constains one or more declarations seperated by semicolons.

Each declaration includes a CSS property name and a value, separated by a colon.

A CSS declaration always ends with a semicolon, and declaration blocks are surrounded with curly braces.

In the following examples all <p> elements will be center-aligned, with a red text color:

Example

p {
  color: red;
  text-align: center;
}

CSS Selectors

CSS selectors are used to "find" (or select) HTML elements based on their element name, id, class, attribute, or more.

The element Selector

You can select <p> elements on a page like this (in this case, all <p> elements will be center-aligned, with a red text color):

Example
p {
  color: red;
  text-align: center;
}

The id Selector

The id selector uses the id attribute of an HTML element to select a specific element.

The id of an element should be unique within a page, so the id selector is used to select one unique element!

To select an element with a specific id, write a hash (#) character, followed by the id of the element.

The style rule bellow will be applied to the HTML with id="para1":

Example
#para1 {
  color: red;
  text-align: center;
}

The class Selector

The class selector selects elements with a specific class attribute.

To select elements with a specific class, write a period (.) character, followed by the name of the class.

In the example bellow, all HTML elements with class="center" will be red and center-aligned:

Example
.center {
  color: red;
  text-align: center;
}
